23 April 2021

ATHATH updated:

  • Venus human traps no longer take damage from being in a cold environment.
  • Venus human traps now take twice as long to die from being in a hot environment.
  • Shoving someone into a person wearing riot gear will no longer knock the person wearing riot gear down.

ArcaneDefence updated:

  • Elder atmosian armor and helmets aren't flashing error messages now. Facehuggers that infest someone don't become pink/black checkerboards on your face. Chameleon kit monkey mind magnification helmets default to a red light, and if you pick the secborg hailer for your mask you get a tape recorder stuffed in your mouth.

Baldies updated:

  • empty versions of graves can now be spawned in.
  • Minor grammar fixes for aquarium content.

Ghilker updated:

  • fixed missing plasma mixer from metastation atmos
  • Thermomachines can now be moved around. To do this, open up the panel with a screwdriver, then Right-Click on the machine with a wrench, after a delay the machine will be un-anchored; use ctrl-click to move the machine around. Then set the rotation of the machine by Left-Click with a wrench, screwdriver to close the panel, and the machine is functional again.
  • thermomachines can change the piping layer while constructed by hitting them when un-anchored with a multitool

Seris02 updated:

  • adds assembly activation, voting and pull request messages to the info chat tab, adds a warning chat class to the bomb activation message

SuperNovaa41 updated:

  • You can no longer buckle yourself through walls, air flaps, and a few other things that shouldn't be buckled through.

Time-Green updated:

  • fixes plumbing components letting some reagent holders not properly delete

celotajstg updated:

  • Ass scans, stamps, UI icons and other .gif and .png images have been optimized to peak efficiency.

jlsnow301 updated:

  • Players can now initiate votes using the new vote panel (assuming CentCom allows it).
  • Admins' voting list is now properly rendered

maxymax13 updated:

  • completely cleaned up all traces of old unwearable caution signs
